# 🐶 Tindog – Responsive Dog Dating Web App

Tindog is a responsive web application inspired by Tinder, designed to help dogs meet new and interesting companions nearby. This project focuses on modern UI design, responsiveness, and clean frontend structure using Bootstrap.

## 💡 Features

- 🐾 Clean and modern landing page design  
- 📱 Fully responsive using Bootstrap grid system  
- 🎯 Navigation bar with smooth section linking  
- ⭐ Features section highlighting app benefits  
- 💬 Testimonials carousel using Bootstrap  
- 📰 Press section with media logos  
- 💰 Pricing section with multiple plans  
- 📢 Call-to-action section  
- 🔗 Social media icons in footer
